## Despliegue üì¶

_Para ejecutar el proyecto es necesario tener el dispositivo virtual en ejecucion_

```

```
_Una vez reconocido el emulador ejecutamos el siguiente comando para ejecutar el proyecto en dispositivo virtual:_
```

```
_Si todo resulto con normalidad ya se esta eejecutando el proyecto en el dispositivo_
